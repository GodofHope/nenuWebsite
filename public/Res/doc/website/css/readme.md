CSS 指层叠样式表 (Cascading Style Sheets)

- 样式定义如何显示 HTML 元素
- 样式通常存储在样式表中
- 多个样式定义可层叠为一

## CSS3

CSS3是最新的CSS标准，被拆分为"模块"。

模块包括：选择器、盒模型、背景和边框、文字特效、2D/3D转换、动画、多列布局、用户。界面
